Repository for the InfrOS Proof Of Concept Human Computer Interaction Team Project utilizing the wireless light sensing controller as its primary form of modality.

Runs on real Wii U retail hardware.


Milestone 1:
- Set up the application development environment
- Deployed a running build of the application on the target hardware
- Implemented a basic pointing and menuing system
